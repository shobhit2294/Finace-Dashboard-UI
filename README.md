# FinTrack — Personal Finance Dashboard

A modern, responsive personal finance dashboard built with **React, TypeScript, Tailwind CSS, and Vite**.
Track income, expenses, visualize spending trends, and gain financial insights.

## Features

### Dashboard Overview

* Summary cards for **Total Balance, Income, and Expenses**
* Balance trend **area chart**
* Spending breakdown **pie chart**
* Financial insights and analytics

### Transactions

* Full transaction list
* Search by description or category
* Filter by transaction type
* Filter by category
* CSV export
* Add, edit, and delete transactions

### Additional Features

* Dark mode toggle
* Local storage data persistence
* Smooth animations with Framer Motion
* Responsive UI for all screen sizes
* Role-based UI (Admin / Viewer)

## Tech Stack

* **React 18**
* **TypeScript 5**
* **Vite 5**
* **Tailwind CSS 3**
* **Recharts**
* **Framer Motion**
* **React Context API**

## Installation

```bash
git clone https://github.com/shobhit2294/Finace-Dashboard-UI.git
cd Finace-Dashboard-UI
npm install
npm run dev
```

## Build for Production

```bash
npm run build
npm run preview
```
